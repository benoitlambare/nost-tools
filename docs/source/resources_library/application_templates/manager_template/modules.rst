.. _manager_template:

Manager Template
================

This template is for an NOS-T manager application, which synchronizes a test case between disaggregated applications. Elsewhere in this documentation you will see references to *managed* appliations, they respond to control messages sent by this or a similar application. Further information on the **Manager** is in the Interface Control Document :ref:`found here <icdManager>`.
